# TCP to HTTP (Boot.dev) 🧙‍♂️🌐

Этот репозиторий содержит мои решения для курса **"TCP to HTTP"** от образовательной платформы [Boot.dev](https://boot.dev). 

В рамках этого проекта мы спускаемся на уровень ниже привычных веб-фреймворков и реализуем протокол интернета (HTTP/1.1) с нуля, используя язык программирования **Go**.

## 🎯 Цели обучения

* **Глубокое понимание HTTP/1.1:** Разбор того, как именно формируются запросы и ответы на уровне текста и байтов, и что реально происходит под капотом при вызове `fetch("google.com")`.
* **Сетевое программирование на Go:** Работа с TCP-соединениями, чтение и запись потоков данных, обработка пакетов.
* **Прокачка инженерных навыков:** Решение нестандартных архитектурных задач и практика в написании надежного, чистого кода.
